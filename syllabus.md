# PEITT523 Cloud Computing

This reading sequence follows the four-module structure in `Syllabus_PEITT523_Cloud Computing.pdf` and uses the chapter list from `chapters.md`.

Students should read the chapters in the order shown under each module. Core readings cover the direct syllabus topics; supporting readings add practice, examples, or wider context.

## Module 1: Cloud Computing Foundations

**Contact hours:** 9

**Syllabus focus:** cloud computing introduction, computing paradigms, service models, deployment models, cloud characteristics, advantages and disadvantages, cloud computing reference architecture, and cloud computing use cases.

### Reading Sequence

1. **Chapter 1: Cloud Computing**
   - Overview of cloud computing and related computing paradigms
   - Cloud service models: SaaS, PaaS, and IaaS
   - Cloud deployment models: private, public, community, and hybrid cloud
   - Characteristics, benefits, and limitations of cloud computing
   - Cloud computing reference architecture
   - Business and technical use cases

**Course outcome supported:** CO1

## Module 2: Virtualization

**Contact hours:** 10

**Syllabus focus:** virtualization introduction, characteristics and importance of virtualization, types of virtualization, virtualization for private cloud services, virtualization architecture, hypervisors, virtualization platforms, tools and mechanisms, physical and virtual clusters, resource management, and data center automation.

### Reading Sequence

1. **Chapter 2: Virtualization**
   - Need and importance of virtualization in cloud computing
   - Network, desktop, application, CPU, memory, and I/O virtualization
   - Virtualization architecture and hypervisors
   - Virtualization platforms and implementation tools
   - Physical and virtual clusters
   - Resource management and data center automation

**Course outcome supported:** CO2

## Module 3: Compute, Storage, and Cloud Security

**Contact hours:** 10

**Syllabus focus:** architectural design of compute and storage clouds, data center interconnection networks, cloud service models and platforms, resource management challenges, fundamental cloud security, security threats, operating system security, infrastructure security, secure distributed data storage, cryptography, cloud computing attacks, data security requirements, security controls, and security architecture.

### Reading Sequence

1. **Chapter 3: Architectural Design of Compute and Storage Clouds**
   - Data center interconnection networks and design
   - Architectural design of computing clouds
   - Cloud service models and platforms
   - Resource management and design challenges

2. **Chapter 4: Secure Distributed Data Storage in Cloud - Methods and Practices**
   - Security issues in cloud environments
   - Cryptography for cloud data protection
   - Cloud data security risks and requirements
   - Cloud security controls and architecture
   - Secure distributed data storage practices

3. **Chapter 12: Cloud Computing Governance and Compliance**  
   Supporting reading for governance, compliance, risk, and policy context related to cloud security.

**Course outcome supported:** CO3

## Module 4: Distributed Programming, Cloud Software, Databases, and Platforms

**Contact hours:** 9

**Syllabus focus:** parallel and distributed programming models, MapReduce, inverted index case study, Twister and iterative MapReduce, Hadoop library, Apache Spark, cloud software environments, popular cloud databases, popular cloud platforms, AWS, Google Cloud Platform, and Microsoft Azure.

### Reading Sequence

1. **Chapter 8: Parallel and Distributed Programming Models**
   - Principles of parallel and distributed programming
   - Programming models used in cloud and big data systems

2. **Chapter 9: Implementation and Testing of MapReduce Programming Model in Cloud**
   - MapReduce programming model
   - Logical data flow and control flow
   - MapReduce case study: inverted index of documents
   - Hadoop-based implementation and testing

3. **Chapter 10: Apache Spark - A Framework for Big Data**
   - Apache Spark fundamentals
   - Spark as a cloud and big data processing framework

4. **Chapter 5: Emerging Cloud Software Environments**
   - Eucalyptus
   - OpenNebula
   - OpenStack
   - Aneka
   - CloudSim

5. **Chapter 6: Popular Cloud Databases**
   - MongoDB
   - HBase
   - Cloud database concepts and use cases

6. **Chapter 7: Big Data Analytics in Cloud**  
   Supporting reading for analytics workflows that use distributed cloud platforms.

7. **Chapter 11: Use Cases: Hadoop, R, and Social Network Analysis**  
   Application-oriented reading for Hadoop, R, and social network analysis use cases.
